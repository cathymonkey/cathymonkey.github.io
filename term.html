Unit 4 Terminology

IP Address Internet Protocol, a unique address that identifies a device on the internet or a local network.

IP version 4 and version 6 IPv4 is the fourth generation of IP, using a 32-bit address. IPv6 is the most recent version of IP, using a 128-bit address. They are not interoperable so it’s impossible to directly communicate with IPv4 and IPv6.

SSH Secure Shell Protocol is a cryptographic network protocol for operating network services securely over an unsecured network.(command-line, login).

SCP Secure Copy Protocol, a means of securely transferring computer files between a local host and a remote host or between two remote hosts.

Terminal commands: LS, CAT, ECHO, CD, CHMOD

LS list all the files in the current directory.

CAT Concatenate, create single or multiple files, view file contents, concatenate files and redirect output or files.

ECHO Outputs the strings it is being passed as arguments.

CD Change the current working directory.

CHMOD Change mode.

Server A server is a piece of software that listens for client requests on a particular set of ports and responds to those requests.

Router A router is a networking device that forwards data packets between computer networks. Routers perform the traffic directing functions on the Internet.

Switch A switch is a device in a computer network that connects other devices together.

LAN Local Area Network, usually confined to one building, using wired communications like Ethernet.

WLAN Wireless LAN, the same as LAN but use wireless technologies
